# Ch 01
Boolean gates 是 Boolean functions（布尔函数）的物理实现

0 和 1 作为布尔值的 false 和 true
Boolean functions 操作二进制输入，然后返回二进制输出

真值表
布尔表达式

- And y
- x Or y
- Not x

Nand 函数可以构建 And, Or, Not ，并且所有的布尔函数都可以又 And, Or, Not 来构建，因此 Nand 可以用来实现硬件内的任何布尔函数（Boolean function）
